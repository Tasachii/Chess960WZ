# Chess960 WZ

## Project Description

- **Project by:** Phasathat Jaruchitsophon (6610545375)
- **Game Genre:** Strategy / Board Game
- **Variant:** Chess960WZ (Fischer Random Chess)

A two-player local Chess960 game built with Python and Pygame. Each game starts from one of 960 randomly generated back-rank arrangements, removing reliance on opening memorization. Every move is recorded into CSV files, exported as a standard PGN, and visualized through an in-app analytics dashboard with seven dynamic chart types.

---

## Installation

Clone the repository:

```sh
git clone https://github.com/Tasachii/Chess960WZ.git
cd Chess960WZ
```

Create and activate a virtual environment, then install dependencies.

**Windows:**
```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

**Mac / Linux:**
```sh
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

---

## Running Guide

After activating the virtual environment:

**Windows:**
```bat
python main.py
```

**Mac / Linux:**
```sh
python3 main.py
```

> The `images/` folder containing the piece sprites must be in the same directory as `main.py`. If sprites are missing the game still runs with text-based placeholders.

## Tutorial / Usage

1. **Main Menu** — choose to play as White or Black, or open History & Stats.
2. **Time Control** — pick Bullet (1 min), Blitz (3 min), Rapid (10 min), or Classical (30 min).
3. **In-Game controls:**
   - Click your own piece to select it. Valid destination squares appear as dots.
   - Click a destination square to move.
   - **Castling (Chess960):** click the king, then click the destination square indicated by the castling marker. The king lands on c-file or g-file and the rook on d-file or f-file regardless of starting position.
   - **Pawn promotion:** the promotion dialog opens automatically when a pawn reaches the last rank.
   - Press **F** to flip the board (useful when two players share one keyboard).
   - Press **R** to return to the main menu mid-game.
   - Click **Resign** under either side label to end the game immediately for that side.
4. **End of game** — result is recorded automatically, a PGN file is written to `pgn_files/`, and you can press **Enter** to start a new game with a fresh Chess960 layout.
5. **History & Stats** — accessible from the main menu. Shows summary statistics and a list of recent games with winners, durations, move counts, and time controls.
6. **Interactive Analytics Dashboard** — open from inside History & Stats. Switch between seven chart types (Piece Dependency, Capture Hesitation, Think Time by Phase, Lethality Matrix, Win Rates, Duration Distribution, Move Trend) and filter by side (White / Black / Both).

## Game Features

- Chess960 random starting position generated by the official 5-step combinatorial algorithm (4 × 4 × 6 × 10 × 1 = 960 unique back-ranks).
- Full chess rule enforcement: legal moves, check, checkmate, stalemate, and the 50-move draw rule.
- Chess960-aware castling (king travels to c- or g-file, rook to d- or f-file; path-clear and not-walking-through-check verified along the king's actual travel path).
- En passant capture.
- Pawn promotion dialog with all four piece options.
- Countdown clock with four time controls (Bullet, Blitz, Rapid, Classical).
- Board flip (press F) for same-keyboard two-player play.
- Live algebraic notation panel beside the board, updating after every move.
- Last-move highlight and pulsing check indicator.
- Material advantage bar showing White vs Black piece-value differential.
- Resign buttons for each side and a game-over overlay with replay option (Enter = new game, R = menu).
- PGN export to `pgn_files/` after every game, formatted with full Chess960 metadata (`[Variant "Chess960"]`, `[FEN "..."]`).
- Persistent statistics across sessions via `statistics/games_history.csv` and `statistics/moves_detail.csv`.
- Interactive Analytics Dashboard with seven dynamic chart types and a side filter.
